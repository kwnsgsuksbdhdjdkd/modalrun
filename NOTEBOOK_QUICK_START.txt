🚀 HOW TO RUN (3 EASY STEPS):

Step 1: Upload Files to Modal Notebook
   ├─ Open your Modal.com notebook
   ├─ Create directory: mkdir -p /root/comfyui-api
   ├─ Navigate: cd /root/comfyui-api
   └─ Upload the files listed above

Step 2: Run Setup (one time)
   bash notebook_setup.sh

Step 3: Start the API
   python start_with_ngrok.py

   OUTPUT:
   ✅ ngrok tunnel created!
   🌍 PUBLIC URL: https://abc123.ngrok-free.app
   📬 Discord notification sent!

   YOU'LL GET:
   - Public HTTPS URL
   - Discord message with the URL
   - API ready to generate images!

🔧 COMMON ISSUES:

Problem: "No models found"
Fix: Run setup script, it will download for you

Problem: "Port already in use"
Fix: pkill -f "main.py" && pkill -9 ngrok

Problem: "ComfyUI won't start"
Fix: Make sure /root/ComfyUI exists and is set up correctly
